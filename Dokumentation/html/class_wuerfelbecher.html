void Wuerfelbecher::wuerfeln ( )

Der Spieler hat 3 Versuche um seine Würfelkombination zu wählen. Im ersten Versuch werden alle 5 Würfel benutzt. Später wird der Benutzer gefragt, ob er weiterhin würfel will oder der Ergebnis aufschreiben will. Am Ende jedes Durchgangs werden die Würfel angezeigt, der Spieler gerade gewürfelt hat (unsortiert) und der Ergebnis dieser Runde (sortiert)
